# Particles
A particle system package for the 2D library [ebiten](https://github.com/hajimehoshi/ebiten).

## Examples

```
go run examples/editor/*.go
```

![example](docs/particles_example.gif)

## Rate over distance

```
go run examples/distance/main.go
```
